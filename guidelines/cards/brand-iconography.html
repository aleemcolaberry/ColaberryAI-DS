Inline icons · .cb-i (size via font-size, color via currentColor)

Icon tiles · .cb-icon-tile (resize with --tile)

RemixIcon (Apache-2.0) is the brand's standard set — prefer the -line style to match our soft 2px look; use -fill for emphasis. Always wrap an icon in <span class="cb-i"> — it guarantees a centered 1em box with no baseline gap. For a colored container use .cb-icon-tile (never hand-roll place-items sizing). Never use emoji.
